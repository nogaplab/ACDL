How to Use

Two ways to use this skill with Claude Code

Option 1: Attach the File

  1. 1 Download acdl-skill.md to your project
  2. 2 In Claude Code, use @acdl-skill.md to attach the file
  3. 3 Ask Claude to generate ACDL for your system

Option 2: Paste Directly

  1. 1 Click the Copy button above
  2. 2 Paste the skill content into your Claude Code prompt
  3. 3 Follow with your request to generate ACDL
