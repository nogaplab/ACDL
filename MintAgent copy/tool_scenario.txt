The scenario is the following:
The user asks the system to book a trip to a specific location.
The rest of the details may be vague, but can be extracted from their calendar, which you are given access to, 
and the .docx document added to the input, which is like notes about their likes and dislikes.

The tools available are:
1. calendar lookup
2. document scan
3. a flight booking assistant (that needs to get budget, dates and location)
4. hotel booking assistant (needs to get budget, location and dates) 
5. ask the user clarifying questions

The pddl will describe the prompts built to deploy such an agent, assuming all the tools exist separately.
